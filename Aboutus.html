About Me

Hi! I’m Maha Lakshmi Nekkanti, currently pursuing my master’s degree in Artificial Intelligence and Business Analytics at the University of South Florida. I’m enrolled in ISM 6225: App Development for Analytics, where I had the opportunity to develop this interactive website.

Through this project, I gained valuable hands-on experience with HTML, CSS, and JavaScript, learning how to build engaging web interfaces and enhance user interactivity. I focused on crafting a consistent and user-friendly design while integrating essential web technologies.

Throughout the development process, I learned how to implement CRUD functionality using localStorage, enabling users to add, view, edit, and delete products seamlessly. I also organized data effectively across different pages to ensure a smooth and cohesive user experience. Additionally, I explored integrating Chart.js to build an interactive analytics page that displays product sales trends in an intuitive and visually appealing way.

This project not only enhanced my technical skills but also deepened my appreciation for the importance of responsive design and user experience. I’m proud of the consistent styling and accessibility of the website, which ensures a positive user journey across devices.

You can find the complete source code for this website on my GitHub Repository.
