About Us

Hello! I’m Maha, the designer behind ChicBoutique. I’m currently studying ISM 6225 – Distributed Information Systems at the University of South Florida. As part of this course, I created this website to showcase my skills in HTML, CSS, JavaScript, and Tailwind CSS.

Through this project, I learned how to implement CRUD functionality using localStorage, and how to organize data effectively on different pages. I also explored integrating Chart.js to build an interactive analytics page that displays sales trends.

This project helped me understand the importance of responsive design and user experience, and I enjoyed building a site that looks great on both desktop and mobile devices. Tailwind CSS made it easier to maintain a consistent style across all pages.

You can view the complete project source code on my GitHub Repository.

Thank you for visiting ChicBoutique—I hope you enjoy exploring it as much as I enjoyed building it!
